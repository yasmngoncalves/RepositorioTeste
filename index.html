Criando meu Repositório no GitHub

Para criar meu repósitorio, usei os seguintes comandos

    

    git init para inicializar o repositório

    git add ArquivoExemplo para colocar o arquivo na área de stagging antes do commit

    commit -m "mensagem" para de fato dar o commit no repositório

    branch -M "main" alterar o nome da branch principal (recomendado)

    Criar o repositório no GitHub

    git remote add origin link-repositorio.git conectar local que meu commit será feito da máquina para o GitHub

    git push -u origin main empurrar o commit para o GitHub
